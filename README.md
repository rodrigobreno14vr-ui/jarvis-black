# Jarvis Black – Site Estático Profissional

Projeto de site estático desenvolvido com foco em performance, simplicidade e baixo custo.
Ideal para sites institucionais, landing pages, blogs e portfólios.

## Sobre o projeto
Este site foi desenvolvido utilizando Hugo, um gerador de sites estáticos rápido e seguro,
eliminando a necessidade de banco de dados ou sistemas pesados como WordPress.

## Tecnologias utilizadas
- Hugo
- HTML5
- CSS3
- Linux
- Git

## Para quem este projeto é ideal
✔️ Pequenas empresas  
✔️ Profissionais autônomos  
✔️ Portfólios pessoais  
✔️ Landing pages  
✔️ Blogs simples e rápidos  

## Diferenciais
- Sem WordPress
- Sem banco de dados
- Carregamento rápido
- Baixo custo de hospedagem
- Estrutura limpa e organizada
